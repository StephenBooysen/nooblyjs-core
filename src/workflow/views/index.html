Workflow Management

Define, start and manage multi-step workflows

Define Workflow

Unique identifier for the workflow
Example steps array:
[
  "/path/to/step1.js",
  "/path/to/step2.js",
  "/path/to/step3.js"
]
User onboarding example:
[
  "/workflows/steps/validate-user.js",
  "/workflows/steps/send-welcome-email.js",
  "/workflows/steps/create-profile.js"
]
Enter a valid JSON array of step file paths. Each step should be a JavaScript file path.

Start Workflow

Name of the workflow to start
User data example:
{"userId": 123, "email": "user@example.com", "plan": "premium"}
Order processing example:
{"orderId": "ORD-001", "customerId": 456, "items": [{"id": 1, "qty": 2}]}
Optional initial data to pass to the workflow. Leave empty if no data is needed.
