Class: GCPFilingProvider

GCPFilingProvider(optionsopt, eventEmitteropt)

A class that implements a Google Cloud Storage-based file storage provider. Provides methods for creating, reading, updating, deleting, and listing files in GCS buckets.

Constructor

new GCPFilingProvider(optionsopt, eventEmitteropt)

Initializes the Google Cloud Storage provider.

Name Type Attributes Description

 GCS bucket name (required).

 GCP project ID (optional, can be from service account).

 Path to service account key file (defaults to './.config/cloud-storage.json').

 Service account credentials object (alternative to keyFilename).
location string <optional>
 Bucket location for creation (defaults to 'US').

 Bucket storage class for creation (defaults to 'STANDARD').
