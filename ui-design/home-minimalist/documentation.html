Installation

Getting started with Noobly JS is straightforward. You can install it using npm or yarn.

bash 
# Using npm
npm install noobly-js

# Using yarn
yarn add noobly-js

Note: Noobly JS requires Node.js version 14 or higher.

Quick Start

Let's create your first Noobly JS application in just a few minutes.

1. Initialize Your Project

bash 
mkdir my-noobly-app
cd my-noobly-app
npm init -y
npm install noobly-js

2. Create Your First Server

javascript 
// app.js
const { NooblyApp } = require('noobly-js');

const app = new NooblyApp({
  port: 3000,
  env: 'development'
});

// Enable caching
app.enableCaching();

// Enable logging
app.enableLogging();

// Basic route
app.get('/', (req, res) => {
  res.json({ message: 'Hello from Noobly JS!' });
});

app.start().then(() => {
  console.log('Server running on port 3000');
});

3. Run Your Application

bash 
node app.js

Success! Your Noobly JS application is now running at http://localhost:3000

Configuration

Noobly JS provides flexible configuration options to customize your application behavior.

javascript 
// noobly.config.js
module.exports = {
  server: {
    port: process.env.PORT || 3000,
    host: '0.0.0.0'
  },
  cache: {
    provider: 'redis',
    config: {
      host: 'localhost',
      port: 6379
    }
  },
  logging: {
    level: 'info',
    format: 'json'
  },
  database: {
    type: 'postgresql',
    connection: {
      host: 'localhost',
      port: 5432,
      database: 'myapp'
    }
  }
};

Caching

Noobly JS provides powerful caching capabilities with support for multiple providers including Redis, Memcached, and in-memory storage.

javascript 
// Enable caching with Redis
app.enableCaching('redis', {
  host: 'localhost',
  port: 6379,
  ttl: 3600 // 1 hour default TTL
});

// Use caching in routes
app.get('/api/users', async (req, res) => {
  const cached = await app.cache.get('users');
  if (cached) {
    return res.json(cached);
  }
  
  const users = await fetchUsersFromDatabase();
  await app.cache.set('users', users, 1800); // 30 minutes
  res.json(users);
});

Data Serving

Efficiently serve and manage data with built-in providers for various data sources.

javascript 
// Configure data serving
app.enableDataServing('simpledb', {
  connection: {
    host: 'localhost',
    port: 5432,
    database: 'myapp'
  }
});

// Auto-generate REST API
app.dataServe('/api/users', {
  table: 'users',
  methods: ['GET', 'POST', 'PUT', 'DELETE'],
  validation: {
    name: 'required|string|max:100',
    email: 'required|email|unique:users'
  }
});
